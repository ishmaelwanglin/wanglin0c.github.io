    打印

    fmt.Print fmt.Printf fmt.Println的区别

    Print(strValue) 打印后边不会换行
Printf("") 完全类似C语言的printf
Println(strValue) 类似java，print的同时后边打印换行符.
Go语言输出printf的时候不能把代码写成多行.

    输出转换说明

    %v 可以输出一切类型的数值，

    %T 可以显示数据类型，

    %b 表示二进制

    其他和c语言大致相同

    单引号字符的数据类型是int32 ,所以在print和println的时候输出都是数字，只能在printf时使用%c来输出字符

    反引号之间的字符串可以原样输出，不需转义。

    str1 :=`
        when i was young
        i just going to radio
        gone
    `
    fmt.Println(str1)

    --------------------
            when i was young
        i just going to radio
        gone

    变量

    全局变量声明后可以不被使用

    局部的变量必须被使用，否则编译报错

    变量的声明方式

    var keyName type
// 批量声明变量
var (
    name string
    age int
    isOk bool
)

// 变量推导: 根据变量的值推导出类型
var keyName = value
// 简短变量的定义，只能在函数里使用
keyName := value

// 匿名变量： 即一个下划线`_`，类似python，该变量接收返回值，但不使用。匿名变量不占用命名空间和不分配内存。

    批量声明常量

    const (
    n1 = 100
    n2 
    n3
)

    下边n2 n3没有赋值就是说 n2 n3都和n1一样 == 100

    iota

    只能在定义常量时使用，变量不能用

    const出现的时候iota被重置为0， 在最近的一个const 处恢复赋值0

    每新增一行const的声明iota加1.

    const (
  _ = iota    //舍弃 iota == 0
  KB = 1 << (10 * iota) // iota == 1
  MB                                        // iota == 2
  GB
  TB
  PB
  YB
)
下边的值不写赋值，说明是和上边的一致

const (
  _ = iota
  n1 = 16
  n2 = iota
  n3,n4
  n7
)
此时n2 == 2 ，n1 行虽然显式赋值， 但是此时iota已然自增
n3和n4在同一行赋值，所以此时n3和n4 都等于 3
n7==4 因为n3和n4在同一行const

    整形变量

    int8 类似char

    int16 类似 short

    int32 类似int

    Int64 类似long

    简短声明n := 10 默认是int类型

    所以尽量使用int8 int16 int32 int64这样的方式声明整形, 如: n:= int8(10)

    uint8 uint16 uint32 uint64 代表无符号型

    特殊整形

    rune :在for index,value range str{ .. } 中，value的数据类型就是rune， 而rune其实就是int32

    byte ：其实质就是uint8，即一个字节,8 bites

    uint和int 具体是站几个bits看系统位数是32还是64

    uintptr ： 指针的地址

    浮点数

    Go语言中默认小数都是float64

    除非声明为float32

    布尔值

    true or false

    不能和整数进行转换，也就是说不能直接用1和0代替

    使用包strings

    import "strings"
strings.Join(数组|切片,连接符)         连接成一个字符串
strings.LastIndex(字符串，匹配的字符)   从右边开始往左匹配到的第一个字符，但是显示的索引值是从左开始数的
strings.Index(字符串, 匹配的字符)                从左边开始匹配到的第一个字符的索引值,索引都是从0开始

strings.HasSuffix(字符串，“匹配的字符”)    字符串是否以匹配的字符结尾，后缀是否是匹配的字符串
strings.Split(str,"sp")                             将字符串用sp切割，得到一个切片

    goto

    goto label 的用法基本上和C一致

    package main

import (
    "fmt"
    "os"
)

func main(){
    var right int = 1
    const MAX = 10
    for ;right < MAX; right++{
        for left := 1;left <= right;left++{
            if left == 2{
                goto out
            }
            fmt.Printf("%d * %d = %d ", left, right, left*right)
        }
        fmt.Println()
    }

    fmt.Printf("before out.")
    out: {
        fmt.Println("\nThis is out label.")
    }
    fmt.Println("after out.")
    os.Exit(0)
}
---
输出如下：
1 * 1 = 1 
1 * 2 = 2 
This is out label.
after out.

    switch

    类似C但是不太一样，没有break；要使用break的特性就得使用fallthrough

    C语言中switch语句：
switch (整型表达式)
case 常量1: 语句;break;
case 常量2：语句;break;
default: 语句;
    Go语言中switch语句:
场景1： 常规的用法
    var num = 10
    switch num{
    case 1: 
        fmt.Println("it is the first")
    case 2: 
        fmt.Println("it is the second")
    case 3: 
        fmt.Println("it is the third.")
    default: 
        fmt.Println("Did not got it.")
    }
---
Did not got it.

场景2：    类似C中 case语句没有写break, 会继续往下执行,直到有break也就是直到没有falltrough下落
    var num = 2
    switch num{
    case 1: 
        fmt.Println("it is the first")
        fallthrough
    case 2: 
        fmt.Println("it is the second")
        fallthrough
    case 3: 
        fmt.Println("it is the third.")
        fallthrough
    default: 
        fmt.Println("Did not got it.")
    }
---
it is the second
it is the third.
Did not got it.

场景3:    swtich后直接大括号，在case语句做判断
var num = 2
    switch {
    case num > 1: 
        fmt.Println("it is the first")
    case num > 2: 
        fmt.Println("it is the second")
    case num > 3: 
        fmt.Println("it is the third.")
    default: 
        fmt.Println("Did not got it.")
    }
---
it is the first

场景4: 变量在switch语句声明， 作用域仅在switch语句内
    switch num := 2 ; {
    case num > 1: 
        fmt.Println("it is the first")
    case num > 2: 
        fmt.Println("it is the second")
    case num > 3:
        fmt.Println("it is the third.")
    default: 
        fmt.Println("Did not got it.")
    }
    或者
    switch num := 2 ; num{
    case 1: 
        fmt.Println("it is the first")
    case 2: 
        fmt.Println("it is the second")
    case 3:
        fmt.Println("it is the third.")
    default: 
        fmt.Println("Did not got it.")
    }

    条件判断

    Go 语言的if语句用法和C完全一致，只是同样条件语句部分不要小括号

    if condition {
    ...
}
else if condition {
    ...
}
else {
    ...
}

    特别注意

    • 所有的语句大括号都得在语句后，连接着写不能换行重写类似ansi C的
    func main(){
    ...
}
if condition{
    ...
}
for {
    ...
}
大括号不能换行
    • 同样小括号里的语句也不能换行
    fmt.Printf( ... , ,,,)
小括号里的内容不能换行
    • 各类索引值从0开始

    • func main(){} 不能带返回值，且没有参数，否则编译报错

      如果要知道退出值是否正常，使用os.Exit(0)

    import "os"
func main(){
    os.Exit(0)
}

    • ! Go语言函数外的变量必须以关键字开头

    • 注释方法和C99一致 // 单行， /* … */ 多行

    1. 编写代码分别定义一个整形、浮点型、布尔型、字符串型变量，使用fmt.Printf()搭配%T分别打印出上述变量的值和类型.

      package main

import(
    "fmt"
    "os"
)

func main(){
    var (
        iNum int32
        fNum float32
        bBol bool
        sStr string
    )
    fmt.Printf("iNum type is %T,\nfNum type is %T,\nbBol type is %T,\nsStr type is %T\n", iNum, fNum, bBol, sStr)
    fmt.Printf("iNum value is %v,\nfNum value is %v,\nbBol value is %v,\nsStr value is %v\n", iNum, fNum, bBol, sStr)
    os.Exit(0)
}
---
未初始化的数值类为0，bool值是false,字符串为"\0"即空字符串，\0为非打印字符，字符结束(C语言)

    1. 编写代码统计出字符串”hello沙河小王子”中汉子的数量 (一个汉字站三个字节即三个索引)

      package main

import(
    "fmt"
    "os"
    "unicode"
)

func main(){
    const str1 = "hello world天津滨海"
    var count = 0
    for _,c := range str1{
        if unicode.Is(unicode.Han, c){
            count++
        }
    }
    fmt.Println("有",count,"个汉字。")

    os.Exit(0)
}
